### 🔥 Featured Projects

#### 🌾 [AnnaData — AI-Driven Agricultural Diagnostic Platform](https://github.com/harshit111singh)
`Python` `CNN` `DeepSeek LLM` `OpenCV`
Custom CNN for real-time crop disease and pest classification from field images, integrated with an LLM to generate contextual agronomic recommendations — built to help farmers protect their crops.

#### ✈️ [Predictive Maintenance — RUL Estimation (NASA C-MAPSS Turbofan Dataset)](https://github.com/harshit111singh)
`Python` `TensorFlow` `LSTM` `Time-Series`
End-to-end predictive maintenance pipeline for turbofan engine degradation using multivariate sensor time-series. Achieved **RMSE 14.64 / MAE 10.69**, outperforming the Zheng et al. LSTM baseline (RMSE 16.14) with a simpler architecture.

#### 😷 [Face Mask Detection — Custom Object Detection Model](https://github.com/harshit111singh)
`Python` `TensorFlow` `OpenCV`
Built a face mask detection model from scratch (no YOLO or pre-built detection libraries) — full detection pipeline engineered using core deep learning and computer vision principles for real-time video and static images.
